# Kamban Agent

## Skills

- Next.js app implementation with App Router, TypeScript and dark UI.
- Shadcn-style component composition with accessible buttons, badges and panels.
- MVVM separation through domain models, application use cases, infrastructure repositories and view models.
- Local markdown Kanban parsing where folders are columns and `.md` files are cards.
- Filesystem card movement by renaming files between column folders.
- Unit testing with Vitest and coverage enforcement above 80%.

## Tools

- Next.js for the web application runtime.
- Tailwind CSS for the dark minimalist interface.
- Shadcn-style local UI primitives in `components/ui`.
- `@dnd-kit` for drag-and-drop card movement.
- `gray-matter` for Markdown/frontmatter-safe parsing.
- Vitest, Testing Library and V8 coverage for unit tests.
- ESLint and TypeScript for static quality gates.
